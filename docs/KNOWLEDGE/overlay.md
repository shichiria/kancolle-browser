# オーバーレイ & 大破警告

## 大破 (Taiha) 検出

- 閾値: `hp / maxhp ≤ 0.25` AND `hp > 0`
- ダメコン判定: 装備に `icon_type == 14` (応急修理) があるかチェック
- `slot` と `slot_ex` の両方をチェック
- 警告トリガー: `/api_req_map/next` (次のノードへ進撃時)

## 陣形記憶

- キーフォーマット: `"{map_area}-{map_no}-{cell_no}"` (例: "1-1-3")
- `formation_memory.json` に保存
- 戦闘開始時に保存、再訪時に呼び出し

## オーバーレイ座標

- ゲームキャンバス基準: **1200×720**
- 全座標にズーム倍率 AND DPI `scale_factor` を適用
- macOSは追加オフセット (+6 dx, +30 dy) が必要 (陣形ヒント用)
- コントロールバー高さ: 28px (キャンバス高さから減算)

## 陣形ボタン配置

- 通常: 3×2 グリッド (陣形ID 1-6)
- 連合: 2×2 グリッド (陣形ID 11-14)
- ハードコードされた中心座標
